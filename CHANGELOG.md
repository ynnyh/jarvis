# 更新日志

记录每个版本的新增、优化和修复。版本号遵循语义化版本（major.minor.patch）。

## Unreleased

## v0.7.0

### 优化
- **CircleCI 发版流水线**：新增 Node.js、Rust、npm、cargo 构建缓存，后续构建大幅加速。
- **Windows 构建资源升级**：executor 从 medium 升级到 large，更多 CPU 和内存。

### 修复
- **CircleCI 签名密钥验证失败**：构建步骤直接读取 CircleCI 项目级环境变量，不再手动 unset 导致私钥丢失。

## v0.6.5

### 修复
- **内部 macOS 开发包错误触发 Apple 公证**：没有 Apple 证书时不再向 Tauri 传 `APPLE_ID` / `APPLE_PASSWORD` / `APPLE_TEAM_ID`，避免打包时报 `Team ID must be at least 3 characters`。

## v0.6.4

### 新增
- **手动写工时**：单页布局，任务按四类（运维 / 事务 / 新增功能 / 其他）分 Tab，运维下额外按项目（任务名前缀）二级筛选，任务支持模糊搜索下拉，工时和内容直接在同一页填写。
- **更新日志**：每次有新版本时弹窗会展示本次更新内容；在「设置 → 关于与更新」里可以查看历史所有版本。

### 优化
- **大窗外观统一**：写工时和手动写工时两个独立窗口去掉系统原生白色标题栏，自定义右上角关闭按钮（hover 红色），与聊天/设置窗口保持一致风格。

### 修复
- **写工时窗口右上角关闭后小人消失、再开模态框无反应**：原生关闭按钮默认销毁窗口实例导致下次找不到 webview，同时关联状态没恢复。改为 Rust 层拦截关闭事件，只隐藏窗口，不销毁。
- **macOS 下载后提示应用已损坏**：发布流水线改为构建 Universal macOS 包；内部开发包新增安装脚本，自动复制到 `/Applications` 并清理 quarantine；更新器同时登记 `darwin-aarch64` 与 `darwin-x86_64`。

## v0.6.3

### 新增
- **聊天渠道接入**：Telegram、QQ Bot 双向消息，未来对话式协作的入口。

## v0.6.2

### 新增
- **Lottie 宠物头像**：替代静态图，动态表情。
- **时段问候**：根据当前时间自动切换问候语和表情。
- **贴边收纳**：小人靠近屏幕边缘时自动收起，留出工作区。

## v0.6.1

### 修复
- 自动更新检查在某些环境下因私有字段访问报错。
- 写工时窗口缺权限配置导致部分操作静默失败。
- 小人在全屏应用上不能被拖动。
- 设置等浮层方向不会根据小人位置自适应。

## v0.6.0

### 新增
- **任务与代码仓库绑定**：commit 自动归属到正确的禅道任务，LLM 智能判断归属。
- **一键写工时**：复盘窗口里看到 commit，弹窗确认就能写入禅道。

### 修复
- 仓库目录绑定时正确展开真实 git 仓库路径。
- 应用业务线排除规则到 commit 列表过滤。
