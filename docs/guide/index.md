# 快速开始

## 安装

前往 [下载页面](/download) 获取适合你操作系统的安装包。

## 首次配置

启动 Jarvis 后，如果配置不完整，会自动弹出**欢迎引导**。

### 1. 禅道配置

填写你的禅道服务器信息：

- **禅道地址**：如 `http://zentao.example.com/zentao`
- **账号**：你的禅道用户名
- **密码**：会加密存储在系统钥匙链，不会明文保存

### 2. 代码目录

配置本地代码仓库路径，用于：
- git 提交扫描
- 代码量统计
- 任务归因
- 工时反推

可以配置多个目录，用分号或逗号分隔：
```
D:/work;D:/projects
```

### 3. LLM 配置（可选）

配置大语言模型服务，开启对话和 Agent 能力：

**推荐配置（DeepSeek）**：
- baseURL: `https://api.deepseek.com`
- model: `deepseek-chat`
- apiKey: 你的 DeepSeek API Key

**其他选择**：
- Moonshot: `https://api.moonshot.cn` + `moonshot-v1-8k`
- Qwen: `https://dashscope.aliyuncs.com/compatible-mode/v1` + `qwen-turbo`
- Claude: 选择 Anthropic 协议，填 `https://api.anthropic.com` + `claude-3-5-sonnet-20241022`

不配置 LLM 也能使用任务管理、工时统计等功能。

## 基础使用

### 桌宠交互

启动后，桌宠会出现在屏幕角落：

- **单击**：展开任务列表
- **右键**：打开菜单（设置/退出）
- **拖拽**：移动位置
- **气泡提醒**：任务到期、逾期时自动弹出

### 快捷键

- `Cmd/Ctrl + Shift + J`：快速唤起主窗口
- `Cmd/Ctrl + ,`：打开设置

可在设置 → 快捷键自定义。

### 任务查询

点击桌宠展开任务列表，或在对话框输入：

```
今天的任务
```

```
逾期的任务
```

```
风险分析
```

### 工时统计

Jarvis 会自动扫描你的 git 提交，按任务归因：

1. 设置 → 代码目录确认配置正确
2. 任务详情里查看「关联 commit」
3. 点击「反推工时」，Jarvis 会按代码量计算建议工时
4. 确认后可一键写回禅道

## 进阶功能

### 对话式操作

配置 LLM 后，可以用自然语言操作：

```
帮我查一下本周完成的任务
```

```
分析一下我逾期的任务风险
```

```
把任务 #123 的工时写回禅道，记 2 小时
```

### 多渠道消息

配置 Telegram 或 QQ Bot 后，可以：
- 随时接收任务到期提醒
- 远程发送命令给 Jarvis
- 白名单控制，只有你能用

配置方式：设置 → 消息通道 → 添加 Bot Token

### 自定义桌宠

设置 → 外观 → 桌宠形象，支持：
- Lottie 动画（推荐，体积小，支持状态）
- PNG/JPG 图片
- GIF 动图

官方提供几套预设，也可以自己导入。

## 数据管理

### 配置文件位置

- **Windows**: `C:\Users\<用户名>\.jarvis\config.json`
- **macOS**: `~/.jarvis/config.json`

可以手动编辑，保存后重启生效。

### 备份

定期备份 `~/.jarvis/` 目录即可，包含：
- 配置文件
- 对话历史
- 日志文件

密钥在系统钥匙链，需要单独导出（或重新配置）。

### 日志查看

日志按天滚动保存在 `~/.jarvis/logs/`，格式：

```
jarvis_2024-01-15.log
```

调试问题时可以查看最新日志文件。
